# Connect Four




### Compilation :
    
Install sdl2 lib :  ``` sudo apt install libsdl-image-dev```
 
make : ``` g++ main.cpp Window.cpp Texture.cpp victory.cpp -w -lSDL2 -lSDL2_image -o Connect_Four ``` 

run : ``` ./Connect_Four```
